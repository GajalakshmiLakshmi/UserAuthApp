# UserAuthServletApp

A simple Java EE web application for user authentication using Servlets and JSP.

## Features
- User Registration
- Login & Logout
- Session Management
- Role-based Access

## Tech Stack
- Java Servlet
- JSP
- Tomcat
- HTML/CSS

## How to Run
1. Import the project in Eclipse
2. Configure Tomcat server
3. Deploy and run on `localhost:8080/UserAuthServletApp`
